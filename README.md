# Quantpilot

股票量化策略回测平台。

Quantpilot 面向量化研究与策略验证场景，目标是构建一个从数据接入、策略开发、回测执行到分析报告与策略管理的完整工作台。平台采用前后端分离架构：前端提供 Web 交互界面，后端负责数据服务、策略执行、回测计算、指标分析与结果持久化。

## 项目概览

项目围绕股票量化研究的完整流程展开，覆盖以下核心方向：

1. 数据接口：统一接入 CSV、数据库、API 和模拟数据，支持日线、分钟线、Tick 等频率。
2. 策略开发与编辑：支持 Python 策略编写、模板复用、技术指标库和参数可视化配置。
3. 回测引擎：采用事件驱动框架，支持撮合、成本模型、账户管理和资金曲线计算。
4. 回测配置与执行：支持时间范围、基准、初始资金、滑点、手续费与批量参数回测。
5. 分析报告系统：输出绩效指标、资金曲线、回撤分析、交易明细和收益分布。
6. 策略管理与分享：支持版本管理、分类组织、导入导出，以及后续协作与策略市场能力。

## 目标功能

### 数据接口

- 支持多种数据源接入：CSV、数据库、API 接口、模拟数据
- 支持多频率数据：Tick、分钟线、日线
- 提供统一数据访问接口，屏蔽底层存储差异

### 策略开发与编辑

- 支持 Python 作为策略开发语言
- 提供代码高亮、自动补全、语法检查
- 内置常用技术指标函数库，如均线、MACD、RSI
- 提供策略模板和示例代码库
- 支持参数可视化配置
- 支持参数优化范围定义与参数敏感性分析

### 回测引擎

- 支持 Bar / Tick 事件驱动机制
- 支持定时事件处理
- 支持多种订单类型，如市价单、限价单
- 支持佣金、滑点、印花税等交易成本模型
- 预留市场冲击模型扩展能力
- 支持账户与资金曲线动态计算

### 回测配置与执行

- 支持回测时间范围选择与滚动窗口回测
- 支持初始资金、基准指数、手续费率、滑点设置
- 支持自定义参数范围和步长
- 支持参数网格搜索
- 支持并行计算与任务队列管理

### 分析报告系统

- 收益指标：年化收益率、累计收益率
- 风险指标：最大回撤、波动率、夏普比率
- 风险调整收益：索提诺比率、卡尔马比率
- 其他指标：胜率、盈亏比、交易次数
- 图表分析：资金曲线、基准对比、回撤曲线、月度热力图、收益分布
- 交易分析：交易明细、买卖点标记、单笔收益分析、持仓时间分布

### 策略管理与分享

- Git 集成或自定义版本管理
- 策略修改历史追踪
- 版本间性能对比
- 策略导入 / 导出
- 策略分类组织
- 策略权限管理、评论和策略市场能力预留

## 技术栈

### 前端

- `React 18`
- `Vite 5`
- `TypeScript`
- `React Router`
- `TanStack Query`
- `Ant Design`
- `ECharts`
- `Monaco Editor`

### 后端

- `Python 3.10+`
- `FastAPI`
- 宿主解耦的事件驱动回测引擎

### 数据与计算

- 当前基础形态：`CSV`、`Mock Data`
- 演进方向：`Parquet` + `DuckDB` + `Polars`
- 兼容分析生态：`Pandas` / `NumPy`

### 实时与任务

- 实时通道：`WebSocket`
- 任务执行：本地任务队列起步，后续可扩展到 `Redis` + `Celery/RQ`

## 本地开发

### 后端启动

```bash
cd backend
python -m venv .venv
. .venv/Scripts/Activate.ps1
pip install -e .[dev]
copy .env.example .env
uvicorn app.main:app --reload --port 8000
```

健康检查地址：`http://127.0.0.1:8000/api/health`

### 前端启动

```bash
cd frontend
npm install
copy .env.example .env
npm run dev
```

默认访问地址：`http://127.0.0.1:5173`  
开发模式下会将 `/api/*` 自动代理到 `http://127.0.0.1:8000`

### 测试

```bash
cd backend
pytest
```

前端可使用以下命令做类型检查：

```bash
cd frontend
npm run typecheck
```
